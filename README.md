# Calculadora de Honorarios para Abogados Perú

Herramienta gratuita para calcular honorarios profesionales en Perú.
Soporta Recibo por Honorarios (4ta categoría), Factura y Boleta de Venta.
Ajustes por complejidad, urgencia y laboriosidad.
